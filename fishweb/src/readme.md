# 海洋生态系统模拟

基于Boids算法的完整海洋生态系统模拟，包含食物链、生长、繁殖和进化机制。

## 项目介绍

这是一个使用HTML、CSS和JavaScript开发的静态网页应用，模拟了一个完整的海洋生态系统。你可以观察不同种类的鱼在海洋中生活、捕食、繁殖和进化的过程。

## 功能特性

- 🐟 多种鱼类，包括草食性、杂食性、肉食性和掠食性鱼类
- 🍽️ 完整的食物链系统
- 📈 鱼类生长、繁殖和进化机制
- 🌅 动态环境变化（时间、光照、温度、水流）
- 🎮 交互式控制（添加鱼、投食、暂停、重置）
- 📊 实时统计数据

## 运行项目

### 方法一：使用Python内置HTTP服务器（推荐）

如果你的系统安装了Python 3，可以使用以下命令启动本地web服务器：

```bash
# 在当前目录启动Python HTTP服务器
python3 -m http.server 8000

# 然后在浏览器中访问 http://192.168.1.107:8000
```

### 方法二：使用Node.js的http-server

如果你的系统安装了Node.js，可以先安装http-server，然后启动服务：

```bash
# 全局安装http-server
npm install -g http-server

# 在当前目录启动http-server
http-server -p 8000

# 然后在浏览器中访问 http://192.168.1.107:8000
```

### 方法三：使用PHP内置服务器

如果你的系统安装了PHP，可以使用以下命令：

```bash
# 在当前目录启动PHP服务器
php -S 192.168.1.107:8000

# 然后在浏览器中访问 http://192.168.1.107:8000
```

### 方法四：使用Ruby内置服务器

如果你的系统安装了Ruby，可以使用以下命令：

```bash
# 在当前目录启动Ruby服务器
ruby -run -ehttpd . -p8000

# 然后在浏览器中访问 http://192.168.1.107:8000
```

## 使用说明

1. 启动web服务器后，在浏览器中打开 http://192.168.1.107:8000
2. 观察海洋中的鱼类活动
3. 使用控制面板的按钮进行交互：
   - `+ 添加鱼`：随机添加一条鱼
   - `🍽️ 批量投食`：在海洋中添加大量食物
   - `🧹 清理食物`：清除所有食物
   - `⏸️ 暂停`：暂停/继续模拟
   - `🔄 重置`：重置整个生态系统
4. 点击海面可以手动投食
5. 观察实时统计数据和生态系统状态

## 技术实现

- 使用HTML5 Canvas绘制海洋和鱼类
- 基于Boids算法实现鱼类群体行为
- 使用JavaScript实现生态系统逻辑
- 响应式设计，适配不同屏幕尺寸
